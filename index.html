• git clone path-to-repository(HTML/SSH)
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль GIT
  • git commit -am "added new file index.html" - зафиксировать изменения и дать комментарий
  • git push - отправить все закомиченные изменения в репозиторий на сервер GIT HUB
  • git pull - забрать свежие обновления с сервера GITHUB
